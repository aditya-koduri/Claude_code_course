üîÑ Query Processing Flow

1
User submits query through web interface
2
FastAPI receives query and forwards to RAG System
3
RAG System creates session (if needed) and retrieves conversation history
4
AI Generator uses search tools to find relevant course content
5
Vector Store performs semantic search on document chunks
6
Retrieved context is sent to Claude AI for response generation
7
Response and sources are returned to user via web interface
